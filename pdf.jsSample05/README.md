# pdf.jsSample04

## 1. 패키지 인스톨

```bash
npm install
```

## 2. 웹팩 빌드

```bash
npm run build
```

## 3. 디버깅

### 3.1. 라이브 서버 디버깅 방법

```bash
# 3.1.1. 웹팩 watch 모드 실행
npm run watch
# 3.1.2. vscode 라이브 서버 실행
# 3.1.3. vscode Live Server Launch Chrome로 디버깅
```

### 3.2. 웹팩 devServer 디버깅 방법

```bash
# 3.2.1. 웹팩 devServer 실행
npm run server
# 3.1.2. vscode webpack devServer Launch Chrome로 디버깅
```
